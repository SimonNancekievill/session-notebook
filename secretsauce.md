## Github

- Cloudbased Hosting Service for Git Repositories(Repos)
- good tool for teamwork
- `isn‘t` git [^1]
- social network for devs and companies

## Markdown

- easy to read markup-language[^2]
- accessable on any device
- human readable

[^1]: Git is an open source version control system that: keeps track of all changes made to the source code, enables developers to easily collaborate on the same project and exchange updates and enables developers to go back to earlier versions of the source code.

[^2]: A markup language is a text-encoding system which specifies the structure and formatting of a document and potentially the relationships among its parts.
